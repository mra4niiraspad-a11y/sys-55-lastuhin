Задание 1
На лекции рассматривались режимы репликации master-slave, master-master, опишите их различия.

Ответить в свободной форме.
 Master‑Slave — один главный сервер и копии для чтения; Master‑Master — несколько равноправных серверов, каждый может и читать, и записывать, но требует более сложной настройки.
